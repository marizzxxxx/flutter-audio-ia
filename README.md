# audios

Exemplo de uso de um modelo para detectar comandos por áudios captados pelo microfone.

O modelo foi treinado usando Teachable Machine e os dados utilizados no treinamento foram: https://drive.google.com/drive/folders/1VyoR1CWO7FecSx3d1dmB82VzlM9Dpp-Q?usp=sharing

Documentação do SoundFlutter: https://pub.dev/packages/flutter_sound/example
